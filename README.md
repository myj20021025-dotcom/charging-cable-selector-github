# 充电站电缆截面选型（静态页）

依据 GB 50217-2018 附录 C、D 的辅助计算页。部署 **GitHub Pages** 后，将公开访问地址发给客户即可在微信中打开。

## 开启 GitHub Pages

1. 本仓库 **Settings → Pages**
2. **Build and deployment**：Source 选 **Deploy from a branch**
3. Branch 选 **main**、文件夹 **/ (root)**，保存
4. 约 1 分钟后访问：`https://<你的用户名>.github.io/charging-cable-selector-github/`

若仓库名不同，请将 URL 中的仓库名改成你的仓库名。
